# TerraFlow

## Overview

TerraFlow is a GSM-based smart irrigation system designed for remote control and monitoring of agricultural water distribution.

## Objectives

- Remote irrigation control
- Low-cost deployment
- GSM connectivity
- Multi-zone valve control

## Hardware

- ESP32
- N58 GSM Module
- Relay Drivers
- Solenoid Valves
